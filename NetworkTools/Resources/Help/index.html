Network Tools Help

Network Tools is a macOS utility for checking interface details, pinging hosts, and scanning ports.

Info Tab

  • Select a network interface from the picker.
  • Review interface information such as address, status, vendor, and model.
  • Review transfer statistics for sent/received traffic and errors.
  • Use copy buttons or right-click context menu actions to copy values.

Ping Tab

  • Enter a destination (IPv4 address or hostname).
  • Choose finite mode or unlimited mode.
  • Use Ping to start and Stop to cancel.
  • Use the copy icon in Output to copy all results.

Port Scan Tab

  • Enter a destination (IPv4 address or hostname).
  • Choose a custom range or enable full scan (1–65535).
  • Use Scan to start and Stop to cancel.
  • Use the copy icon in Output to copy all results.

Tips

  • If start buttons are disabled, confirm destination/range input is valid.
  • Some interface details depend on hardware and driver support.
  • Scan and ping results depend on network conditions, firewall rules, and host settings.
