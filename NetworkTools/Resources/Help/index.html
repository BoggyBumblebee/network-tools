Network Tools Help

Quick Start

  1. Open the tab for the task you need: Info, Ping, or Port Scan.
  2. Enter a destination host when required (IPv4 or hostname).
  3. Run the action using Ping or Scan.
  4. Use Stop at any time to cancel an active run.

Info Tab

  • Use the interface picker to select a network interface.
  • Interface rows show link state, addressing details, and hardware metadata when available.
  • Transfer statistics refresh automatically and are formatted with locale grouping.
  • Some values can be unavailable if the driver or hardware does not expose them.

Ping Tab

  • Destination must be a valid IPv4 address or hostname.
  • # Pings to Send applies in finite mode; unlimited mode runs until stopped.
  • Output shows progress, timing lines, and a clear termination reason.
  • If the button is disabled, check destination format and ping count validity.

Port Scan Tab

  • Scan supports a custom port range or the full range (1–65535).
  • Progress and open-port results stream into the output panel.
  • Scans can be stopped at any time; completion lines summarize scanned and open counts.
  • Results depend on host availability, firewall policy, and network routing.

Copying Data

  • Use copy controls on output panels to copy all output text.
  • Use per-field copy controls on supported Info rows for quick value export.
  • Text fields that are selectable can also be copied via standard macOS copy actions.

Troubleshooting

Buttons are disabled

Verify destination syntax and any numeric range inputs.

No ping or scan results

Check host reachability, DNS resolution, and firewall/network policy.

Interface values are missing

Some vendors/drivers do not expose all hardware fields to the operating system.

Safety and Privacy

  • Run scans only on systems and networks you own or are authorized to test.
  • Output may include hostnames, addresses, and port data; treat copied output as sensitive.
  • Use smaller scan ranges first when testing unfamiliar hosts.

Accessibility

  • The app supports keyboard navigation across tabs, fields, and controls.
  • Primary actions expose disabled states so invalid inputs are clear before execution.
  • Help is reachable from the Help menu and can be opened quickly with Command + Shift + /.
